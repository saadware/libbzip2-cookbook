# libbzip2-cookbook

Installs the libbzip2 development package.

## Supported Platforms

* CentOS
* RHEL
* Debian
* Fedora
* Ubuntu

## Usage

### libbzip2::default

Include `libbzip2` in your node's `run_list`:

```json
{
  "run_list": [
    "recipe[libbzip2::default]"
  ]
}
```
